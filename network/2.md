# 信道的极限
## 信息
信息用来消除不确定性，信息代表的事件发生的概率越小信息量越大

信息量的定义：`I = log(1/P(X)) = -logP(X)`

## 奈奎斯特极限
根据奈奎斯特采样定理而来
无干扰理想情况（连分子热运动都不能有）

## 香农极限
信道的香农极限（或称香农容量）指的是在会随机发生误码的信道上进行无差错传输的最大传输速率。

`C = B * log(1 + S / N)`，其中`N=n * B`，`n`为噪声功率
